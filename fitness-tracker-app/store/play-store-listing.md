## Screenshots Required

Google Play requires **2–8 screenshots** per device type.

### Phone Screenshots (16:9 or 9:16, min 320px, max 3840px)
Capture these key screens:
1. **Activity Tracking** — The main tracking screen with map, metrics, and controls
2. **Activity Complete** — Post-activity summary with stats
3. **History** — Activity history list showing past activities
4. **Statistics** — Charts and analytics view
5. **Goals** — Goal setting and progress screen
6. **Profile** — User profile with personal records
7. **Dark Mode** — Any screen in dark mode

### Feature Graphic (1024 x 500 px)
- Required for Play Store listing
- Show the app name "Stride" with the tagline and a device mockup

### App Icon (512 x 512 px)
- Already have this: `assets/icon.png`
- Ensure it meets Google Play's icon guidelines (no transparency, no rounded corners — Play Store rounds them)
