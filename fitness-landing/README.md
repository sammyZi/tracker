## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
```

This will create an optimized static export in the `out` directory.

## Sections

- **Hero**: Eye-catching introduction with call-to-action
- **Features**: Key app features with icons
- **Stats**: User engagement metrics
- **Download**: App store links and benefits
- **Footer**: Contact and copyright information

## Customization

- Update colors in `tailwind.config.ts`
- Modify content in `app/page.tsx`
- Add your app store links in the download section
- Replace placeholder stats with real metrics
